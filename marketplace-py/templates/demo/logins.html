Technical
Stack and deployment

Backend

Django 5.2, Django REST Framework, SQLite (dev). Custom user model with roles, job lifecycle state machine with auto-transitions, Open Payments SDK for Interledger.

Frontend

Server-rendered templates with HTMX. WCAG-accessible with semantic HTML, ARIA labels, and skip links. Custom audio player system with language fallback chain.

Audio System

Generic audio snippet model (attaches to any content via ContentType). Community contribution workflow. Language fallback chain with Otomi-specific fallback audio.

Payments

Open Payments protocol with GNAP authorization. Contract lifecycle: create incoming payment, get quote, authorize via wallet redirect, complete outgoing payment.

Reload demo: make full-demo
